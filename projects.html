To-do List

Vanilla JavaScript, CSS, HTML

To-do List which is custom designed and coded with vanilla JavaScript. Data is persisted to local storage. Dynamically create and delete tasks, customize tasks with titles, descriptions and priorities. Tasks are sorted and color coded based on their priority levels - low, important or urgent. Check tasks when they're completed. Incomplete task count is updated whenever tasks are completed. Functionality to clear completed tasks or clear all tasks. Responsive for mobile.
